# NaiSell Client

## Project Purpose

NaiSell is a modern second-hand marketplace platform where users can buy and sell pre-owned products easily and securely. The platform helps reduce waste, encourages sustainable shopping, and creates opportunities for users to earn money from unused products.

## Live URL

🌐 https://your-client-live-url.com

## Key Features

- User Authentication with Better Auth
- Google Login Support
- JWT Protected Routes
- Responsive Design (Mobile, Tablet, Desktop)
- Dynamic Product Listings
- Product Search & Sorting
- Product Categories
- Wishlist System
- Buyer Dashboard
- Seller Dashboard
- Admin Dashboard
- Stripe Payment Integration
- Dark/Light Theme Toggle
- Framer Motion Animations
- Skeleton Loading UI
- Custom 404 Page
